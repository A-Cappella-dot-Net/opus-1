# test-presto

Standalone test and benchmark applications for [presto](../presto/README.md). Functional test apps cover SnS (snap and subscribe), the supported field types, FT membership and monitoring, and a minimal pipe/sink client–server pair. Performance harnesses (ping-pong, burst, drop, pub/sub) measure latency and throughput, reporting percentile statistics. Launched with the same bin/config layout as the deployable applications.
